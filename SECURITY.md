# Security Policy

## Supported Versions

This project is maintained as a single active version in the main branch.

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |

## Reporting a Vulnerability

If you discover a security issue, please do not disclose it publicly before it has been reviewed.

Please report vulnerabilities by contacting the project maintainer privately and include:

- A clear description of the issue
- Steps to reproduce
- The potential impact
- Any suggested remediation, if available

You should expect:

- An acknowledgment as soon as possible
- A review of the report
- A fix or mitigation if the issue is confirmed

## Scope

This policy applies to:

- The React/Vite application source code
- Static assets shipped with the site
- Client-side content and configuration committed in this repository

This policy does not cover third-party platforms or services linked from the website.
